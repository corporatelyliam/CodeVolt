BRPLL — Pool League Manager

A full league-management platform built for a pool lounge: player and account management, tournaments (round-robin and knockout), multi-season leagues with automatic promotion/relegation, division championships, and singles/doubles team leagues — all reflected on a live leaderboard the moment a score is entered.

Up to three admins can score games from separate devices at the same time without conflicts, thanks to atomic score updates and a real-time sync layer — changes appear on every connected screen instantly. Players get their own login to check standings, upcoming fixtures, and career stats, while admins get full control over accounts, branding, and dashboard announcements.

React 18 + Vite Node.js + Express SQLite (better-sqlite3) Docker Cloudflare Tunnel Server-Sent Events
